Protocol Buffers 是一种轻便高效的结构化数据存储格式，可以用于结构化数据串行化，或者说序列化。它很适合做数据存储或 RPC 数据交换格式。可用于通讯协议、数据存储等领域的语言无关、平台无关、可扩展的序列化结构数据格式。JSON是一种基于JavaScript语法的轻量级数据交换格式，它独立于语言与平台，具有自我描述性，易于理解，便于解析与生成，优秀的特性使得JSON被越来越多的使用
