ubuntu 16.04 server 安装和部署

发表于 2018-04-15 | 更新于 2018-04-16 | 分类于 动手实战 | 评论数：



Ubuntu 是基于Debian GNU/Linux，分为桌面版和服务器版，Ubuntu每六个月便会发布一个新版，以便人们实时地获取和使用新软件。Ubuntu共有五个长期支持版本(Long Term Support，LTS)：Ubuntu 6.06、8.04、10.04、12.04、14.04、16.04。

安装ubuntu 16.04 server的过程，留作备份

一、ubuntu server 16.04 LTS 系统安装

  1. 首先是刻录镜像到u盘这个暂且不说，教程很多。推荐用软碟通去刻录到u盘中，然后重启电脑，选择u盘启动即可进入安装界面
  2. 出现语言语言选择界面，我们选择English ,回车进入下一步
  3. 默认选择第一项:Install Ubuntu Server，回车再弹出界面中选择时区 –> Hong Kong
  4. 选择键盘布局所属国家(keyboard) –> English（US），等待预加载完毕
  5. 选择网络配置(network）–> 默认回车即可
  6. 设置主机名(hostname) –> 自由设置，默认为ubutnu
  7. 设置用户名(username) –> 自由设置，作为登录使用
  8. 设置密码设置(password) –> 自由设置，可不填写
  9. 选择是否加密home目录(encrypt home directory) –> No
  10. 选择时区是否正确 –> 正确选Yes，不正确选No手动选择
  11. 设置磁盘分区(parition disks) –> Manual
  12. 选择磁盘 –> 选要选择的磁盘以SCSI开头
  13. 选择新建一个空白分区表(create a new empty parition table on this device) –> Yes,选择磁盘空闲空间FREE SPACE
  14. 选择新建分区(create a new partition) –> 输入空间大小new partition size –> 选择Primary –> 选择头部建立(Biginning) –> 设置资料系统file system –> 设置挂载点Mount point –>点击分区设定完毕Done seeting up the partition,反复三次，分别建立以下3个分区
    | 挂载点名称 | 文件格式 | 空间大小 |
    | —— | —– |——– |
    | /boot | ext4 | 400MB |
    | /swap | swap | 内存大小的1.5-2倍 |
    | / | ext4 | max |
  15. 选择结束分区并写入磁盘
  16. 设置网络代理HTTP proxy –> Continue
  17. 设置不自动更新No automatic update
  18. 选择需要安装的软件Choose software to install –> 空格为选定
  19. 安装GRUB boot loader –> yes
  20. 完成安装重启

二、 安装完成后的一些配置

  1. 登录系统，输入设置好的username 和 password

  2. 启用root账号来提升权限

  3. 查看当前网络状态，默认网卡为en开头那块

  4. 编辑配置网络，服务器需要固定其ip来让外部用户使用，位于/etc/network下，文件名为interface

  5. 修改The primary network interface 下的内容保存后推出

  6. 重启网络，应用配置，若未生效，则需要重启服务器
